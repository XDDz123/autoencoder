# Autoencoder
This project implements convolutional autoencoders in Python using PyTorch with CUDA acceleration. </br>
The models are train on the MNIST and CelebA-HQ datasets. </br>
## Libraries
* numpy
* torch
* cv2
* os
* pathlib
* torchvision
* matplotlib
